
web-image-gen

  Modern responsive web image generation and tooling.                           
                                                                                
  Default config merged with `.web-image-gen.js` or `.json`.                    
                                                                                
  https://github.com/brev/web-image-gen/tree/main/packages/cli                  

  Usage:   `web-image-gen <command> [options ...]` 

Available Commands

  generate    Generate responsive images and importable code manifests.   
  originals   Perform helpful operations on original source images.       
  clean       Remove all generated images and manifests.                  
  help        Show this usage help guide, or help for a specific command. 

Global Options

  -c, --config string   Config file to use instead of the default, in `.js` or  
                        `.json` format.                                         
  -h, --help            Show this usage help guide, or help for a specific      
                        command.                                                
  -v, --verbose         Display debugging output.
