
Originals

  Perform helpful operations on original source images.        
                                                               
  https://github.com/brev/web-image-gen/tree/main/packages/cli 

  Usage:   `web-image-gen originals [options ...]` 

Originals Options

  -p, --optimize    Minimize size of original source images while trying to     
                    maintain quality.                                           
  -r, --remove      Remove original source images (shrink size of production    
                    release package).                                           
  -f, --force       Bypass confirmation prompt when using `--remove` option.    

Global Options

  -c, --config string   Config file to use instead of the default, in `.js` or  
                        `.json` format.                                         
  -h, --help            Show this usage help guide, or help for a specific      
                        command.                                                
  -v, --verbose         Display debugging output.
